How to use Google Maps Vendor Finder for HBloom:

1. Save vendor JSON file as Database/partner_data.json
2. Run python script database/convert_json.py
3. Open templates/searchPage.html in Google Chrome or Safari (Firefox is okay but IE does not seem to work)
4. Enter client address and distance range. Click "Search"
5. Filter vendors based on preferences
6. Click on the map pin of the preferred vendor to find their contact information
7. Call the vendor and order the flowers!
